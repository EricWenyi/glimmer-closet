# Glimmer Closet - Telegram 使用指南

## 👋 欢迎使用智能衣柜！

你可以通过 Telegram 直接管理你的衣柜。

## 📸 上传新衣服

直接发送：
1. **衣服照片**（单张或多张）
2. **文字描述**（可选）：
   - 衣服名称
   - 分类：上衣/裤子/裙子/外套/鞋子/配饰/包包
   - 颜色
   - 适用季节
   - 适用场合

**示例消息：**
```
黑色羊绒大衣
分类：外套
颜色：黑色
季节：秋冬
场合：工作、正式
```

## 🔍 查询衣柜

发送指令：
- `查看衣柜` - 显示所有衣服
- `查看外套` - 按分类筛选
- `查看黑色衣服` - 按颜色筛选
- `找春季衣服` - 按季节筛选

## 📋 衣服编码

每件衣服都有唯一编码：**C-XXXXXX**
查询时可以用编码快速找到特定衣服。

## 💡 小贴士

- 照片会自动保存到云端
- 系统会自动识别颜色和分类（可手动修正）
- 随时可以通过网页查看：http://localhost:3001
